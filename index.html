HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan menandai konten pada halaman web.

HTML memberikan kerangka dasar untuk merancang dan memformat halaman web, memungkinkan penyusunan teks, gambar, hyperlink, formulir, dan elemen-elemen lainnya.

Bahasa markup adalah jenis bahasa pemrograman yang digunakan untuk menandai atau memberi instruksi kepada teks.

Dalam konteks HTML, ‚Äùmarkup" mengacu pada tag dan elemen yang digunakan untuk memberi petunjuk tentang bagaimana sebuah halaman web seharusnya diorganisir dan ditampilkan.

Berikut adalah beberapa contoh penggunaan HTML:
  • Heading
  • Paragraf
  • Pendukung Paragraf
  • Teks Tebal
  • Teks Italic
  • Teks Underline
  • Teks Coret
  • Hyperlink
  • Image/Gambar
  • Ordered List
  • Unordered List
  • Tabel
  • Merge Cell Tabel (Rowspan & Colspan)
  • Form
  • Label dan Submit

CSS

CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah.

Dengan CSS, kita dapat mengatur layout (tata letak), warna, font, garis, background, animasi, dan lain-lain.

Saat ini CSS terbaru adalah versi 3

Berikut adalah beberapa contoh penggunaan CSS:
  • Inline CSS
  • Internal CSS
  • Eksternal CSS
  • Selector CSS
  • Color
  • Background
  • Text Align
  • Text Decoration
  • Text Transform
  • Font Size
  • Font Weight
  • Font Style
  • Font Family
  • Border
  • Border Sisi Tertentu
  • Sizing (Width & Height)
  • Spacing (Margin & Padding)
  • Display
